## 第一章：平台概述与核心概念

### 1.1 平台定位与设计理念

火宝短剧是一款专注于AI短剧生成的综合性创作平台，其核心设计理念是**降低短剧创作的技术门槛**，让创作者能够专注于故事本身的表达，而非繁琐的技术实现。传统的短剧制作流程涉及剧本撰写、角色设计、分镜绘制、视频剪辑等多个专业领域，通常需要团队协作完成。而火宝短剧通过整合多种AI能力，将这一复杂流程简化为几个简单的操作步骤，让个人创作者也能独立完成高质量短剧的制作。

平台的设计遵循三个核心原则：**易用性优先**、**创意为核心**、**渐进式增强**。易用性优先意味着每一个功能的设计都以降低用户认知负荷为目标，界面布局清晰直观，操作流程简洁明了。创意为核心意味着平台的所有功能都服务于创作本身，不会让技术细节干扰创作者的思路。渐进式增强意味着用户可以先使用基础功能快速产出，随着技能提升逐步探索高级功能，每一次升级都能带来新的创作可能性。

理解这三个设计原则对于高效使用平台至关重要。许多用户之所以无法充分发挥平台能力，往往是因为没有理解功能的**设计意图**，仅仅机械地执行操作而忽略了背后的创作逻辑。接下来的章节中，我们会经常回到这些设计原则，帮助你在理解「怎么做」的同时，也理解「为什么要这样做」。

### 1.2 核心功能模块

火宝短剧平台由五个核心功能模块组成，每个模块对应短剧制作流程中的一个关键环节。这五个模块分别是：**剧本管理**、**角色系统**、**分镜制作**、**视频生成**和**资源库**。理解这五个模块之间的关系和各自的职责，是高效使用平台的基础。

| 模块名称 | 核心功能 | 在工作流程中的位置 | 关键操作 |
|----------|----------|-------------------|----------|
| 剧本管理 | 剧本的创建、编辑、版本管理 | 短剧创作的起点 | AI生成、手动输入、模板应用 |
| 角色系统 | 角色的创建、形象生成、素材管理 | 剧本之后的角色准备阶段 | AI生成形象、上传图片、角色关联 |
| 分镜制作 | 剧本到视觉的转化、镜头设计 | 连接剧本与视频的桥梁 | AI自动生成、手动调整 |
| 视频生成 | 分镜到视频的转化、特效添加 | 最终产出阶段 | 参数配置、批量生成 |
| 资源库 | 资产管理、素材整理、导入导出 | 贯穿整个工作流程 | 分类管理、标签标注、批量操作 |

这五个模块之间存在明确的**数据依赖关系**：剧本定义了角色和场景的需求，角色为分镜提供视觉素材，分镜为视频生成提供基础画面。这种依赖关系决定了工作流程的基本顺序：先有剧本，再有角色，然后是分镜，最后是视频。但这个顺序并非绝对僵化——平台支持在任意阶段回溯修改，修改会自动影响到后续环节的生成结果。

### 1.3 术语表与概念定义

为了确保后续章节的描述清晰无歧义，本节定义了平台中使用的核心术语及其含义。这些术语将贯穿整个手册，请在开始学习之前建立对这些概念的基本认知。

**剧本（Script）**：是短剧创作的核心文档，包含场景描述、角色对白、动作指导等元素。剧本不仅定义了「说什么」（对白），还定义了「做什么」（动作）和「在哪里做」（场景），是整个短剧的信息载体。平台的AI能力能够从简单的创意描述自动生成结构完整的剧本。

**角色（Character）**：是短剧中出现的虚构人物实体，包含人物形象（视觉表现）和人物设定（文字描述）两个维度。角色与剧本中的对白和动作相关联，确保同一角色在不同场景中的形象一致性。平台支持通过AI生成角色形象，也支持上传自定义图片。

**分镜（Storyboard）**：是将剧本文字转化为视觉画面的中间产物，包含场景图像、镜头类型、画面构图等信息。分镜定义了「最终视频的每一帧是什么样的」，是连接文字剧本和视频产出的关键桥梁。一个完整的短剧由多个分镜组成，每个分镜通常对应3-10秒的视频片段。

**镜头（Shot/Camera Angle）**：是分镜画面的拍摄视角定义，包括特写、近景、中景、全景、俯拍、仰拍等类型。镜头语言是影视创作的重要表达手段，通过不同的镜头选择可以传达不同的情感和信息。例如，特写适合强调角色情感，全景适合交代环境氛围。

**提示词（Prompt）**：是用户向AI系统传达需求的自然语言描述。提示词的质量直接决定了AI生成结果的好坏。好的提示词应该**具体、明确、有指导性**，避免模糊和矛盾的要求。平台的AI功能都提供了提示词优化建议，帮助用户写出更好的提示词。

**本地路径（Local Path）**：是平台生成的资源文件在本地存储系统中的相对路径。平台采用「先本地后云端」的存储策略，所有生成的文件默认保存在本地存储目录中，通过URL路径访问。这种设计确保了数据的安全性和可控性。

## 第二章：剧本管理系统

### 2.1 剧本创建方法

剧本是整个短剧创作的起点，平台提供了三种创建剧本的方法：**AI辅助生成**、**模板应用**和**手动输入**。这三种方法各有适用场景，理解它们的特点可以帮助你选择最合适的创作方式。

**AI辅助生成**是平台最具特色的功能之一。用户只需提供一个简单的故事创意或主题关键词，AI就能自动生成完整的剧本结构。这种方法的**核心优势**是快速——从创意到完整剧本可能只需要几分钟；**核心局限**是AI生成的剧本可能不够个性化，需要用户进行修改和润色。AI辅助生成特别适合以下场景：快速验证故事创意、获取创作灵感、生成剧本初稿供后续修改。

AI辅助生成的工作原理是这样的：首先，用户输入的故事创意被发送到文本生成AI（如GPT、Claude等）；然后，AI根据创意解析出故事的核心要素（主角、冲突、解决等）；接着，AI按照标准的剧本格式组织内容，生成场景描述、对白、动作指导等完整元素；最后，生成的剧本返回给用户进行审核和修改。整个过程中，AI的「思考」是基于大量剧本数据学习到的模式，它理解什么样的故事结构更吸引人，什么样的对白更自然流畅。

**模板应用**是另一种高效的剧本创建方式。平台内置了多种类型的剧本模板，涵盖常见的短剧类型：浪漫喜剧、悬疑推理、都市情感、历史穿越、科幻未来等。使用模板的**核心优势**是结构保证——模板提供了经过验证的故事结构，用户只需要填充具体内容；**核心局限**是创意受限——模板会限制故事的走向和可能性。模板特别适合以下场景：学习剧本结构、保证产出质量、快速产出标准化内容。

**手动输入**是最灵活但也最耗时的方式。用户可以直接在剧本编辑器中输入任何内容，不受AI生成或模板格式的限制。这种方法的**核心优势**是完全控制——你可以精确实现任何想要的效果；**核心局限**是效率较低——需要投入更多时间和精力。手动输入适合以下场景：有明确的创作意图、需要精确控制每一个细节、修改AI生成的剧本。

**方法选择决策树**：

```
我想创建剧本：
├── 我有清晰的故事构思
│   ├── 我想要AI帮我扩展和完善 → 使用AI辅助生成
│   ├── 我想学习剧本结构 → 使用模板应用
│   └── 我想完全控制 → 手动输入
├── 我只有一个简单的创意
│   └── 使用AI辅助生成（最快速有效）
└── 我需要快速产出标准化内容
    └── 使用模板应用（质量有保证）
```

### 2.2 剧本结构详解

专业的剧本有其独特的格式和组织方式，理解这些结构有助于你更好地使用AI辅助生成功能，也便于在手动输入时产出规范的内容。火宝短剧采用的剧本格式融合了专业剧本规范和AI友好的特点，既能被AI系统正确理解，也符合行业的阅读习惯。

**剧本的基本组成单元**是「场景（Scene）」。每个场景定义了特定的**时间**（日/夜/晨/昏）、**地点**（室内/室外/具体位置）和**人物**（参与场景的角色）。场景是剧本的逻辑划分单位，一个完整的短剧通常包含5-20个场景。场景与场景之间通过「转场」进行连接，常见的转场方式有：淡入淡出、切换、叠化等。

**场景的标准格式**如下所示：

```markdown
场景编号：[场景1]

【场景设定】
时间：内景 / 外景
地点：具体位置描述
时段：日 / 夜 / 晨 / 昏
主要人物：角色A、角色B

【场景描述】
（此处描述场景的环境、氛围、角色状态等客观信息）

【动作与对白】
角色A：（情感/动作）对白内容
角色B：（情感/动作）对白内容
...
```

理解这个格式后，你会发现AI辅助生成的本质是**模式填充**——AI学习了很多这样的剧本模式，当你提供故事创意时，AI识别出需要的故事元素，然后按照上述格式「填入」合适的内容。这就是为什么提示词越具体详细，AI生成的结果越符合预期——因为更多的输入约束意味着更少的「猜测空间」。

**剧本层级结构**由大到小依次为：剧本→幕（Act）→场景（Scene）→段落（Beat）。短剧通常采用**三幕结构**（Three-Act Structure），即：
- **第一幕（建置）**：介绍角色、设定、冲突开端
- **第二幕（对抗）**：主角面对障碍、冲突升级
- **第三幕（解决）**：冲突达到高潮、问题解决

虽然短剧的篇幅通常不支持过于复杂的多幕结构，但理解三幕结构有助于组织故事的节奏感。AI辅助生成时，如果你能在提示词中说明故事处于哪个阶段（例如「这是一个关于主角成长的故事，目前处于冲突升级阶段」），AI生成的内容会更加贴合你的需求。

### 2.3 AI剧本生成技巧

AI剧本生成的效果很大程度上取决于用户提供的**提示词（Prompt）质量**。好的提示词能够让AI准确理解你的意图，生成符合预期的结果；模糊的提示词则可能导致AI「误解」你的需求，产出需要大量修改的内容。本节将系统性地介绍如何写出高质量的剧本提示词。

**提示词的基本要素**包含以下几个维度：

| 要素 | 说明 | 示例 |
|------|------|------|
| 故事类型 | 故事的题材和风格 | 青春校园、悬疑推理、浪漫喜剧 |
| 主线剧情 | 故事的核心事件和发展 | 「主角意外获得超能力后...」 |
| 角色设定 | 主要角色的性格特点 | 「小明是一个内向但善良的高中生」 |
| 情感基调 | 整体氛围和情感走向 | 温馨感人、紧张刺激、轻松幽默 |
| 故事时长 | 预期的剧本长度 | 3-5分钟的短剧，约10个场景 |

**高级提示词技巧**进一步提升了生成控制能力：

**技巧一：使用「否定约束」排除不想要的内容**。AI生成有时会出现「过度发挥」的情况，加入一些你不想要的元素。通过明确说明「不要出现XX」可以有效控制生成范围。

**示例**：```
请生成一个关于友情的青春校园故事，
不要出现任何暴力或悲伤的情节，
角色数量控制在3-5人，
场景主要在学校和家里。
```

**技巧二：提供「参考风格」帮助AI理解审美偏好**。如果你有喜欢的电影、电视剧或小说，可以将其作为风格参考告诉AI。

**示例**：```
故事风格参考《那些年，我们一起追的女孩》，
温馨中带有一点遗憾，
对白风格轻松自然，
但关键时刻有感人的情感表达。
```

**技巧三：分阶段生成而非一次生成完整剧本**。对于复杂的创作需求，一次性生成完整剧本往往效果不佳。更好的做法是分段生成：先生成故事大纲，确认后再逐个场景生成详细内容。

**工作流程示例**：
1. 使用提示词「生成一个关于职场新人的喜剧短剧大纲」→ 获得故事大纲
2. 审核大纲，如有需要修改提示词重新生成 → 确定故事结构
3. 使用提示词「根据大纲，生成第1-3场景的详细内容」 → 获得具体场景
4. 重复步骤3直到完成所有场景

**技巧四：利用「迭代优化」逐步改进**。AI生成很少一次就能达到完美，更现实的做法是先生成「够用」的版本，然后通过追加指令进行优化。

**示例**：```
第一轮生成后，发现角色对白不够自然，
追加提示词：「请将上述场景中的对白改写得更口语化，加入更多生活化的表达。」
```

### 2.4 剧本版本管理

创作过程中经常需要修改剧本，平台的版本管理功能可以帮助你追踪修改历史、比较不同版本、恢复到之前的状态。版本管理是创作过程中**防错**的重要工具——当你修改出问题，可以随时回退到之前的正确版本。

**版本保存机制**：每次点击「保存」按钮时，系统会自动创建一个新的版本快照。版本快照包含完整的剧本内容、时间戳和修改说明（可选）。系统默认保留最近20个版本，更早的版本会被自动清理。如果你有重要的历史版本需要长期保存，建议导出到本地文件。

**版本比较功能**：在版本历史列表中，选择任意两个版本，系统会高亮显示它们之间的差异。这个功能特别适合以下场景：回顾创作过程、理解AI生成结果的演变、确认最终的修改内容。

**版本恢复操作**：选中任意历史版本，点击「恢复到该版本」按钮，系统会将该版本的内容恢复到当前编辑区。**注意**：恢复操作不可撤销，恢复后当前编辑区的内容会被覆盖。如果你不确定是否要恢复，建议先导出当前内容作为备份。

**最佳实践建议**：

1. **重要节点保存**：在大规模修改之前，先手动保存一个版本快照
2. **版本说明**：填写版本说明描述本次修改的目的，便于后续识别
3. **定期清理**：手动删除不需要的历史版本，避免版本列表过于冗长
4. **本地备份**：对于重要的创作项目，定期导出剧本到本地文件

## 第三章：角色管理系统

### 3.1 角色设计原则

在影视创作中，角色是承载故事情感的核心载体。一个成功的角色应该具备三个特征：**辨识度**（观众能够快速区分不同角色）、**一致性**（角色的言行符合其性格设定）、**成长性**（角色在故事中有所变化和发展）。理解这三个特征，有助于你在设计角色时做出正确的决策。

**辨识度**首先体现在视觉形象上。角色的发型、服装、体态等外观特征应该是独特的，让观众一眼就能认出「这是谁」。其次体现在声音上——虽然短剧主要是视觉媒体，但角色的说话方式、用词习惯也应该有独特之处。AI生成角色形象时，这些辨识度特征会直接影响生成结果，所以描述越具体，辨识度越高。

**一致性**是角色设计的核心挑战。一个角色在故事开头和结尾应该保持基本的性格特征，但又要经历合理的成长变化。一致性不仅仅是「性格不变」，更重要的是「言行符合性格」——一个内向的角色不会突然表现得外向，除非故事中有导致这种变化的合理事件。AI生成角色时，你提供的描述越详细，AI越能维持这种一致性。

**成长性**是让角色「活起来」的关键。静态的角色是扁平的，动态的角色才是立体的。设计角色时，应该考虑：「这个角色在故事中会经历什么变化？」、「冲突会如何改变这个角色？」、「故事结束时，这个角色有什么不同？」这些问题的答案定义了角色的成长弧线。

### 3.2 AI角色形象生成

角色形象生成是平台的特色功能之一。用户只需提供角色的文字描述，AI就能生成对应的视觉形象。这个功能大大降低了角色设计的门槛——即使你不会绘画，也能获得专业水准的角色形象。

**AI形象生成的原理**：当用户提交角色描述后，系统首先使用大语言模型（LLM）对描述进行**语义理解**，提取关键视觉特征（发型、眼睛、服装等）；然后，将这些视觉特征转化为图像生成模型的**提示词**；最后，图像生成模型（如Stable Diffusion、DALL-E等）根据提示词生成角色形象图片。整个过程通常需要30秒到2分钟。

**输入描述的质量**直接影响生成结果。以下是一些编写高质量角色描述的技巧：

**技巧一：从整体到局部**。先描述角色的整体印象（大年龄、性别、体态），再逐步细化到局部特征（发型、眼睛、服装）。这种结构化的描述方式有助于AI系统更准确地理解你的需求。

**示例模板**：
```
[年龄]岁的[性别]角色，
整体气质：[活泼/沉稳/神秘等]
发型：[颜色]的[长短]发，[特殊造型]
面部：[脸型]，[眼睛特征]
服装：[颜色]的[款式]，[配饰]
特殊标记：[胎记/疤痕/纹身等，如果适用]
```

**技巧二：使用具体的形容词而非抽象概念**。抽象概念容易产生多种解释，而具体形容词的目标更明确。

**对比示例**：
```
❌ 抽象：时尚的衣服
✅ 具体：白色oversizeT恤搭配浅蓝色牛仔裤，脚穿白色帆布鞋

❌ 抽象：温柔的眼神
✅ 具体：杏仁眼，眼神柔和，眼角微微带笑
```

**技巧三：控制特征数量**。过多的特征描述可能导致AI无法同时满足所有要求，反而降低了生成质量。建议核心特征控制在5-8个，次要特征可以在后续调整中补充。

### 3.3 角色图片上传与管理

除了AI生成，平台还支持**上传自定义图片**作为角色形象。这个功能适用于以下场景：用户已经有完整的角色设计、希望使用真实人物照片、想要混合AI生成和真人图片。

**上传图片的技术要求**：

| 参数 | 要求 | 说明 |
|------|------|------|
| 文件格式 | JPG、PNG、WebP | 推荐使用PNG以获得最佳质量 |
| 图片尺寸 | 最小512×512像素 | 推荐1024×1024或更高 |
| 文件大小 | 最大10MB | 超过限制会被压缩处理 |

**图片预处理**：上传后，系统会自动对图片进行预处理，包括：中心裁剪为正方形（确保AI生成的一致性）、调整分辨率、添加默认背景。如果需要更精细的调整，可以在上传后使用平台的图像编辑功能进行微调。

## 第四章：分镜制作系统

### 4.1 分镜的概念与作用

**分镜（Storyboard）** 是影视创作中的核心工具，是将文字剧本转化为视觉语言的桥梁。专业的分镜通常是一系列简化的图画，配合文字说明，展示每个镜头应该如何拍摄。在传统制作流程中，分镜由专业的分镜师绘制，耗时数周甚至数月。火宝短剧的AI分镜生成功能将这个过程自动化，让创作者可以在几分钟内获得完整的分镜方案。

理解分镜的作用需要从它的**功能定位**说起。分镜不是「画出来的剧本」，而是「视觉化的剧本」。剧本告诉你「发生了什么」，分镜告诉你「看起来是什么样子」。这种转化涉及多个决策：场景应该是什么样的？镜头从什么角度拍摄？画面中有什么元素？光线如何处理？这些决策共同决定了最终视频的视觉风格和观感。

**AI分镜生成的工作原理**：系统首先分析剧本内容，识别出每个场景的关键信息（地点、时间、出现角色、关键动作）；然后，根据这些信息生成画面描述（提示词）；接着，图像生成模型根据画面描述生成对应的静态图片；最后，系统将生成的图片与场景信息组合，形成完整的分镜。整个过程中，最关键的是「信息转化」——如何将剧本文字中的抽象信息转化为图像生成的明确指令。

### 4.2 AI自动生成分镜

平台的核心功能之一是**AI自动生成分镜**。这个功能能够将剧本自动解析为一系列分镜，每个分镜包含画面图像和详细描述。掌握这个功能的使用技巧，可以大大提高创作效率。

**触发生成分镜**：在剧本编辑页面，确认剧本内容完整后，点击「AI生成分镜」按钮。系统会分析剧本结构，将其拆分为独立的场景单元，然后依次生成分镜。整个过程是**批量处理**的，你不需要逐个场景操作。

**生成过程中的信息展示**：
- **进度条**：显示整体处理进度
- **当前处理**：显示正在处理的具体场景
- **预览窗口**：显示已生成的分镜预览
- **统计信息**：显示已成功/失败的分镜数量

**影响生成质量的因素**：

1. **剧本质量**：剧本描述越详细具体，生成的分镜越准确。「小明走进教室」比分镜生成的结果可能只是「普通教室」；「小明走进一间光线昏暗的旧仓库，墙上斑驳的油漆诉说着岁月的痕迹」则能生成更有氛围感的画面。

2. **角色关联**：如果分镜中涉及的角色已经在角色库中创建，AI会参考角色的形象进行生成，确保同一角色在不同分镜中的一致性。强烈建议在生成分镜前完成主要角色的创建和形象确定。

3. **场景描述**：剧本中的场景描述信息会直接影响分镜画面。添加具体的视觉细节（颜色、物体、光线等）可以显著提升生成效果。

### 4.3 分镜编辑与调整

AI生成的分镜是**起点而非终点**。根据创作需求，你可能需要对分镜进行各种调整。本节介绍分镜编辑的各项功能。

**基本编辑功能**：

| 功能 | 操作说明 | 使用场景 |
|------|----------|----------|
| 编辑描述 | 点击分镜，进入编辑模式修改文字描述 | 优化画面提示词 |
| 更换图像 | 点击「重新生成」按钮重新生成图像 | 对当前图像不满意 |
| 调整镜头 | 在参数面板中选择镜头类型 | 改变画面视角 |
| 调整构图 | 拖拽画面元素调整位置和比例 | 微调画面布局 |

**镜头类型选择**：

| 镜头类型 | 英文 | 视觉特点 | 适用场景 |
|----------|------|----------|----------|
| 特写 | Close-up | 聚焦角色面部或重点物体 | 强调情感、突出细节 |
| 近景 | Medium Close-up | 展示角色上半身 | 对话场景 |
| 中景 | Medium Shot | 展示角色膝盖以上 | 日常动作 |
| 全景 | Wide Shot | 展示角色全身和周围环境 | 交代场景氛围 |
| 大远景 | Extreme Wide Shot | 展示广阔的空间 | 强调环境、宏大场面 |
| 俯拍 | High Angle | 摄像机从上方拍摄 | 俯视效果、弱化角色 |
| 仰拍 | Low Angle | 摄像机从下方拍摄 | 仰视效果、增强气势 |

**镜头语言的基本原则**：

镜头选择不是随意的，而是服务于叙事需求。理解镜头语言的基本原则有助于做出正确的选择：

1. **情感表达**：强烈的情感（惊讶、愤怒、悲伤）通常使用特写，让观众看清角色的表情变化
2. **信息交代**：需要观众了解环境信息时使用全景或大远景
3. **关系暗示**：两个角色同框时，中景和近景适合表现亲密关系，远景适合表现疏离关系
4. **节奏控制**：快速切换镜头产生紧张感，长镜头产生平静感

### 4.4 分镜预览与顺序调整

分镜生成完成后，可以通过**预览功能**查看完整的分镜序列。预览模式模拟了最终视频的观看体验，可以帮助你评估整体效果。

**预览操作**：

| 退出 | 退出预览模式 | Esc键 |

**顺序调整**：

分镜的顺序决定了最终视频的叙事逻辑。在预览或列表视图中，可以通过**拖拽**调整分镜的顺序。拖拽时，系统会显示分镜将被插入的位置，松开鼠标完成放置。

**调整顺序的注意事项**：

1. **时间线连续性**：确保调整后故事的时间线仍然是连贯的
2. **逻辑连贯性**：确保调整后场景转换是合理的
3. **视觉效果**：考虑画面之间的切换是否自然流畅

**分镜拆分与合并**：

- **拆分**：如果某个分镜包含的内容过多，可以将其拆分为多个连续分镜，每个处理原分镜的一部分
- **合并**：如果相邻分镜内容关联紧密，可以合并为一个分镜，生成更长的视频片段

## 第五章：视频生成系统

### 5.1 视频生成原理

视频生成是将静态分镜图片转化为动态视频的过程。火宝短剧平台采用**图像到视频（Image-to-Video）**的生成技术，结合**运镜控制**和**过渡效果**，产出连贯的短视频片段。

**核心技术流程**：

1. **图像分析**：系统分析输入的分镜图片，识别画面中的主体、背景、运动元素
2. **运动预测**：基于图像内容和用户配置的参数，预测画面中元素的运动轨迹
3. **帧生成**：生成指定帧率（FPS）的中间帧，填补原始图像到目标图像的过渡
4. **编码输出**：将所有帧合成为最终的视频文件

**理解这个原理的意义**在于帮助你做出更好的参数配置。例如，当你了解「运动预测」这一步时，就能理解为什么描述剧烈运动的提示词可能产生不稳定的画面——AI在「想象」不存在的运动轨迹时存在局限性。选择相对静态的画面作为分镜，通常能获得更稳定的视频效果。

### 5.2 生成参数配置

视频生成的质量和风格在很大程度上取决于参数配置。本节详细介绍各项参数的作用和推荐设置。

**基础参数**：

| 参数名称 | 可选值 | 默认值 | 说明 |
|----------|--------|--------|------|
| 分辨率 | 720p/1080p/4K | 1080p | 最终视频的清晰度，越高越清晰但生成时间越长 |
| 帧率 | 24fps/30fps/60fps | 24fps | 每秒帧数，影响画面流畅度 |
| 时长 | 3-10秒 | 5秒 | 单个分镜视频的时长 |
| 视频码率 | 2-20Mbps | 8Mbps | 视频质量，越高越清晰但文件越大 |

**运镜参数**：

| 运镜类型 | 效果描述 | 适用场景 |
|----------|----------|----------|
| 静态 | 画面不动 | 对话场景、静态展示 |
| 平移-左到右 | 摄像机从左向右移动 | 展示空间从左到右 |
| 平移-右到左 | 摄像机从右向左移动 | 展示空间从右到左 |
| 推进 | 摄像机逐渐靠近主体 | 强调重点、紧张感 |
| 拉远 | 摄像机逐渐远离主体 | 展示全局、从局部到整体 |
| 环绕 | 摄像机围绕主体旋转 | 立体展示主体 |

**运镜选择的思考框架**：

选择运镜类型时，应该问自己几个问题：

1. **这个场景的核心信息是什么？** 如果是展示角色表情，静态或轻微推进更合适；如果是展示空间关系，平移更合适。

2. **这个场景的情感基调是什么？** 静态适合平静、对话；运动适合紧张、兴奋；缓慢的运镜适合抒情、回忆。

3. **与前后分镜的关系如何？** 运镜应该与叙事节奏匹配，连续的快节奏场景不宜使用缓慢的运镜。

**过渡效果**：

当多个分镜视频拼接成一个完整视频时，分镜之间的过渡效果决定了观感的流畅度。

| 过渡类型 | 效果描述 | 使用建议 |
|----------|----------|----------|
| 硬切 | 瞬间切换 | 快速节奏、紧张氛围 |
| 淡入淡出 | 渐变过渡 | 时间流逝、场景转换 |
| 叠化 | 两画面重叠后切换 | 梦境、回忆、并列叙事 |
| 擦除 | 一个画面被另一个「擦除」 | 空间转移 |

视频生成完成后，可以在平台内进行预览和导出操作。

## 第八章：最佳实践

### 8.1 创作流程最佳实践

高效的创作流程能够显著提高产出质量。基于大量用户的使用经验，我们总结出以下最佳实践：

**实践一：先规划后创作**

在开始任何AI生成之前，先用简单的文字梳理清楚：
- 故事的核心主题是什么？
- 主要角色有谁？
- 故事的高潮在哪里？

这种「先想清楚」的习惯能够避免在创作过程中迷失方向。AI生成虽然快速，但修改的成本也不低——与其生成后大幅修改，不如一开始就规划清楚。

**实践二：善用迭代优化**

不要期望AI一次生成就能达到完美效果。将AI视为「初稿生成器」，然后通过迭代逐步优化：
- 第一轮：生成基础框架或初稿
- 第二轮：针对不满意的部分重新生成
- 第三轮：微调细节，统一样式

这种「多轮迭代」的方式比「一次到位」的期望更现实，也更容易获得满意的结果。

**实践三：建立素材库**

角色、图片、场景等元素一旦创建就应该保存到素材库中：
- 好的角色设计可以在多个项目中复用
- 风格统一的图片素材便于后期统一调整
- 完整的素材库是长期创作的基础设施

### 8.2 提示词优化技巧

提示词是控制AI输出的核心工具。以下是经过验证的优化技巧：

**技巧一：结构化提示词**

将提示词组织成清晰的结构，便于AI理解和执行：

```
【任务类型】
生成一个青春校园短剧场景

【核心要素】
- 地点：学校图书馆
- 时间：下午放学后
- 人物：小明（内向高中生）、小红（活泼同桌）

【场景描述】
阳光透过窗户洒进图书馆，
小明坐在角落专注看书，
小红悄悄走近想要吓他一跳

【风格要求】
温馨、轻松
```

**技巧二：提供正面和负面约束**

正面约束告诉AI「要什么」，负面约束告诉AI「不要什么」：

```
正面约束：生成一个明亮、色彩丰富的场景
负面约束：不要出现暗色调，不要有模糊效果
```

**技巧三：使用参考示例**

如果有你喜欢的风格或效果，可以将其作为参考：

```
请参考以下风格生成画面：
[参考图片]
风格特点：柔和的光线、温暖的色调、居中构图
```

### 8.3 质量控制检查清单

在提交AI生成任务之前，检查以下项目可以提高成功率：

**剧本检查**：

| 检查项 | 合格标准 |
|--------|----------|
| 场景描述 | 每个场景有具体的视觉描述 |
| 角色设定 | 主要角色有明确的形象描述 |
| 逻辑连贯 | 场景之间的转换合理 |
| 长度合适 | 篇幅符合预期产出时长 |

**角色检查**：

| 检查项 | 合格标准 |
|--------|----------|
| 描述详细 | 有足够的外观细节 |
| 风格一致 | 多个角色的风格统一 |
| 命名清晰 | 角色名称易于区分 |
| 关联正确 | 角色与剧本匹配 |

**分镜检查**：

| 检查项 | 合格标准 |
|--------|----------|
| 场景完整 | 覆盖剧本的所有场景 |
| 镜头合理 | 镜头选择符合叙事需求 |
| 描述准确 | 画面描述与文字一致 |
| 顺序正确 | 分镜顺序符合剧情发展 |

## 练习任务

### 练习1：剧本创作练习（⭐⭐）

**任务目标**：独立创作一个完整的短剧剧本

**任务要求**：

1. 主题：选择你感兴趣的主题（青春、悬疑、喜剧等）

| 场景描述 | 25% | 场景有具体的环境细节 |
| 对白质量 | 20% | 对白自然流畅，符合角色特点 |
